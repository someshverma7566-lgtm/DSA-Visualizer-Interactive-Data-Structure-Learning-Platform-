OOP Concepts

Encapsulation, Inheritance, Polymorphism, Abstraction

Encapsulation

Encapsulation means binding data and methods together inside a class.

class Student {
private:
int marks;

public:
void setMarks(int m){
marks = m;
}
};

Inheritance

Inheritance allows one class to use properties of another class.

class Animal {
public:
void eat(){}
};

class Dog : public Animal {
};

Polymorphism

Same function behaves differently.

class Shape {
public:
virtual void draw(){}
};

Abstraction

Hiding implementation details.

class Shape {
virtual void area() = 0;
};
